Este proyecto consiste en el desarrollo de una aplicación bancaria con un diseño inspirado en el estilo característico de Carl’s Jr.. Aunque se trata de una app bancaria funcional y completa, hemos incorporado la identidad de la marca para ofrecer una experiencia única.
Al utilizar esta aplicación, los usuarios podrán acceder a beneficios exclusivos como descuentos, promociones especiales y preferencias en productos y servicios asociados a Carl’s Jr..

Funcionalidades principales:
La aplicación requiere que los usuarios inicien sesión con su correo electrónico y contraseña. Si no cuentan con una cuenta, tendrán la opción de registrarse proporcionando los siguientes datos:

Nombre y apellido
Correo electrónico
Contraseña
Una vez registrados, podrán empezar a disfrutar de las funcionalidades y beneficios exclusivos de Banco Carl’s Jr..
